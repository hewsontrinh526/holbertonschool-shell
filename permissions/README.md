Shell, permissions

0. [My name is Betty](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/0-iam_betty) - Creating a script that switches the current user to the user Betty.

1. [Who am I](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/1-who_am_i) - Creating a script that prints the effective username of the current user.

2. [Groups](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/2-groups) - Creating a script that prints all the groups the current user is part of.

3. [New owner](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/3-new_owner) - Creating a script that changes the owner of a file to the user Betty.

4. [Empty!](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/4-empty) - Creating a script that creates an empty file called hello.

5. [Execute](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/5-execute) - Creating a script that adds execute permission to the owner of the file hello.

6. [Multiple permissions](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/6-multiple_permissions) - Creating a script that adds execute permission to the owner and the group owner, and read permissions to other users, to the file hello.

7. [Everybody!](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/7-everybody) - Creating a script that adds execution permission to the owner, the group owner and the other users, to the file hello.

8. [James Bond](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/8-James_Bond) - Creating a script that sets specific permissions of the file hello.

9. [John Doe](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/9-John_Doe) - Creating a script that sets specific permissions of the file hello.

10. [Look in the mirror](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/10-mirror_permissions) - Creating a script that sets the mode of the file hello the same as olleh's mode.

11. [Directories](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/11-directories_permissions) - Creating a scripts that adds execute permission to all subdirectories of the current directory for the owner, the group owner and all other users while regular files will not have their permissions changed.

12. [More directories](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/12-directory_permissions) - Creating a script that creates a directories with permissions 751 in the working directory.

13. [Change group](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/13-change_group) - Creating a script that changes the group owner to school for the file hello.

14. [Owner and group](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/14-change_owner_and_group) - Creating a script that changes the owner to vincent and the group to staff for all the files and directories in the working directory.

15. [Symbolic links](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/15-symbolic_link_permissions) - Creating a script that change the owner and group owner of _hello to vincent and staff respectively.

16. [If only](https://github.com/hewsontrinh526/holbertonschool-shell/blob/master/permissions/16-if_only) - Creating a script that changes the owner of the file hellow to vincent only if it is owned by the user guillaume.
